Add New Face

Enter the name of the person you want to add. A folder will be created for them in the 'dataset' directory. You then need to manually place their face images (e.g., JPG, PNG) into that new folder (e.g., `dataset/John_Doe/`).

After adding images, remember to click the "Train Model Now" button on the Dashboard!

Future Enhancement: Live Image Capture

In a more complete system, this page would allow you to capture images directly from the webcam for a new person. This feature is not implemented in this basic UI version but is highly recommended for ease of use.

Example flow for live capture:

  1. Enter name and click 'Start Capture'.
  2. Webcam stream appears.
  3. System automatically captures N images of the person's face.
  4. Images are saved to `dataset/Person_Name/`.
